CS-Script
Visual Studio 2013-15 CS-Script Tools extension
CS-Script Tools extension allows managing C# scripts with Visual Studio 2012-15.
The detailed information on how to use this extension can be found in the Online Tutorial.
 
  
For the latest CS-Script distributables visit www.csscript.net.
This extension requires CS-Script (min v2.8.0) to be installed.

Visual Studio 2010 version is also available from the Visual Studio Gallery.
There was an installation problem report on VS2013 due to the .NET 4.5 being in-place upgraded into 4.6. As work around v4.5 reinstallation can be attempted if you experience the same problem. 
Features
  • Create new script
  • Open existing script 
  • Refresh script project: update list of referenced assemblies and included files by parsing the CS-Script directives.
  • Open script by dropping the script file to the extension window.
  • Force CS-Script directives (e.g. //css_reference) to be rendered as grayed out text in the code window.
  • Maintain Recent Scripts list
  • Quick links/commands (e.g. configuration console, dedicated directories, documentation)
